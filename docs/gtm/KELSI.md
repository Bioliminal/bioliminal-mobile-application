# Kelsi Andrews

## Role

Mobile, CV Integration, Capture UX

## What I build

The premium user experience. I integrate the computer vision models into a
Flutter app that fuses live CV with real-time sEMG data from a wearable
sensor — and makes it feel like you're just opening an app and training.

BlazePose at 30 fps for live skeleton overlay. BLE integration with an ESP32
streaming muscle activation data. Session capture, upload, and results — all
in a flow that doesn't ask the user to think about signal chains or sensor
fusion. Aaron trains the models. I make them work in the app.

## Why it matters

The hardest problem in this product isn't any one technical layer. It's making
the seams between them invisible. Camera capture, BLE pairing, live overlay,
haptic feedback, session history — the user shouldn't have to understand any
of it. They should just train.

A closed-loop training system that feels complicated is a closed-loop training
system nobody uses. The interface is where the science either reaches the user
or doesn't.

## What I bring

Frontend developer focused on clean, modern applications designed around user
needs. I integrate the CV models Aaron trains into the mobile app and build
the capture UX on top. The underlying stack is powerful — multi-channel sEMG,
server-side biomechanics ML, haptic actuators, rule-based reasoning. My job is
to make all of that feel like a single, effortless experience.

Premium UX isn't decoration. It's the difference between a demo and a product.
